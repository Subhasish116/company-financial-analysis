 Financial Analytics Capstone Project

## 📌 Project Overview
This project focuses on analyzing financial performance using **SQL, Python, and Power BI**.  

The goal of this project is to clean, validate, analyze, and visualize financial data to identify:

- Revenue trends
- Expense patterns
- Profitability
- Budget vs Actual performance
- Financial efficiency
- Business insights for decision making

This project demonstrates an end-to-end **Data Analytics workflow** from database creation to dashboard visualization.

---

# 🎯 Business Problem

Organizations generate large amounts of financial data but often struggle to:

- Track revenue and expenses
- Measure profitability
- Compare budget vs actual spending
- Identify financial inefficiencies
- Make data-driven decisions

This project solves these challenges through structured data analysis and visualization.

# 🗂 Database Design

The database contains **5 financial tables**:

### 1. Customers
Stores customer-related information.

### 2. Vendor
Contains vendor and supplier information.

### 3. Headcount
Tracks employee and workforce-related data.

### 4. Budget
Contains planned financial budget information.

### 5. Financial Transactions
Stores actual business financial transactions.
